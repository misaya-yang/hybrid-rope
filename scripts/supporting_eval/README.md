# Supporting Evaluation Scripts

These scripts are still useful, but they are not the primary Phase 8–15 experiment chain.

- `eval_longbench.py`: generic LongBench generation-style evaluator
- `eval_multi_needle.py`: standalone multi-needle evaluator for from-scratch checkpoints
- `eval_niah_heatmap.py`: heatmap-style NIAH evaluation for HF-style models
- `eval_niah_recall.py`: recall-style NIAH evaluation for HF-style models
- `eval_passkey_scratch.py`: passkey mix utilities and standalone passkey eval for from-scratch models
