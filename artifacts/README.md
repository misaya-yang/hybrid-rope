# artifacts 目录说明

本目录按机器/日期组织实验产物，是主数据入口。

## 子目录

- `a100_2026-02-13/`：A100 主线实验（50M/100M/350M）
- `a800_2026-02-13/`：A800 Llama-3-8B Hybrid-LoRA 实验
- `r6000_2026-02-13/`：R6000 Qwen2.5-7B Hybrid-LoRA 实验

## 每个机器目录通用结构

- `scripts/`：执行脚本
- `data/` 或 `results/`：JSON/日志等产物
- `figures/`（若存在）：图表
- `README.md`：机器侧实验摘要与入口

## 统一约定

- 不包含模型权重
- 所有路径均可直接用于复现/复核
