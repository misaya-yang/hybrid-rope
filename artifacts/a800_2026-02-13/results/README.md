# A800 results 目录说明

本目录保存 A800 侧的运行结果与日志。

## 子目录

- `llama3_hybrid_lora/`：LoRA 训练结果
- `llama3_hybrid_lora_eval/`：base vs hybrid 评测
- `h800_parallel/`：频率并行搜索结果
- `h800_3h_followup/`：3 小时窗口的定向重训+稳健性复评
- `h800_3h_poly_followup/`：poly 频谱补充重训+稳健性复评
- `mechanism_p1/`：机制验证阶段 P1 全量结果
- `mechanism_p2_framework/`：统一机制框架聚合结果与图表
- `unified_search/`：统一搜索 B split 结果
