# DiT 频率分配理论分析 (v2 — 实验验证后更新)

> **v2 更新 (2026-03-16)**: 基于 head-to-head 实验结果，Power-Shift 族已被否定，
> Cosh 族 τ=1.5 确认为 129.6M DiT 最优配置。"容量依赖"叙事已被推翻。

## 1. 问题背景

EVQ-Cosh 在 AR 模型上已证明有效：通过将部分高频资源挪向低频端，牺牲短距离精度换取长距离外推能力。τ* = K/√T 是 AR 的最优浓度参数。

~~但在 DiT (bidirectional attention) 的 129.6M 实验中，τ=2.83 的 EVQ 被 GEO 大幅击败（denoising precision 差 45-71%）。~~

**v2 修正**: 跨 run 的 GEO 优势完全是 CUDA 非确定性噪声。Head-to-head 验证表明 EVQ-Cosh(τ=1.5) 在 129.6M DiT 上赢 GEO **-21% (train) / -35% (far-extrap)**。

核心问题：**DiT temporal 维度的最优频率分配是什么？**
**答案**: EVQ-Cosh(τ≈1.5)，即 γ≈0.53 的 AR 最优值。

## 2. AR vs DiT：RoPE 频率的不同角色

### 2.1 AR（Causal Attention）

RoPE 频率服务于**信息传播**：

- Causal mask 限制了注意力方向，信息只能从左到右传播
- 低频分量让注意力在长距离上衰减更慢，支撑远距离信息传递
- EVQ 的低频增强直接提升了传播距离和外推能力
- 中频损失可以接受——causal mask 本身就限制了注意力范围

优化目标：最大化注意力核 A(Δ) = Σ_k cos(θ_k·Δ) 在距离 L 处的幅值

最优解：τ*_AR = K/√T（大τ，激进低频增强）

### 2.2 DiT（Bidirectional Attention）

RoPE 频率同时服务于**两个目标**：

**目标 A — 频谱匹配**

视频时间维度变化慢，相邻帧高度相关。temporal power spectrum 集中在低频。RoPE 需要更多低频分量来自然编码这种慢变化。

→ 需要低频增强（方向与 AR 相同）

**目标 B — 位置指纹（Bidirectional 特有需求）**

没有 causal mask，所有帧互相可见。RoPE 是模型区分不同帧的唯一线索。中频段是区分"中等距离帧对"（Δ=5-20）的关键。

→ 中频不能被掏空

**关键矛盾：AR 不在乎中频（causal mask 代劳），DiT 必须保住中频。**

τ=2.83 对 DiT 的失败原因：为了低频增强而过度抽空中频，破坏了 fingerprinting。
τ=1.5 的成功原因：温和的低频增强，中频保留足够。

## 3. 实验验证结果

### 3.1 Head-to-head τ sweep (129.6M DiT, YaRN)

| τ | Train MSE (vs GEO) | Far-extrap (vs GEO) | Winner |
|---|-----|-----|--------|
| 0.30 | worse | worse | GEO |
| 0.70 | ~5x worse | worse | GEO |
| 1.20 | ~2.8x worse | worse | GEO |
| **1.50** | **-21%** | **-35%** | **EVQ** |

**发现：τ=1.2 → τ=1.5 之间存在离散相变。**

### 3.2 Power-Shift 族 (~~路线 B~~, **已否定**)

| α | Train MSE (vs GEO) | 结论 |
|---|-----|--------|
| 0.25 | **22x worse** | 灾难性失败 |
| 0.50 | **6x worse** | 严重失败 |

**结论：Power-Shift 族在实践中完全不可行，已放弃。** 尽管理论上它保护了中/高频，但实际表现远不如 GEO，遑论 Cosh。

### 3.3 跨 run 非确定性影响

所有先前的跨 run 比较均受 CUDA 非确定性影响：

| 比较 | 跨 run 结论 | Head-to-head 结论 | 差异 |
|------|-----------|-----------------|------|
| 129.6M GEO vs EVQ(2.83) | GEO 赢 71% | 待验证 (可能仍是 GEO) | 幅度不可信 |
| 129.6M GEO vs EVQ(1.5) | τ sweep 结果不一致 | EVQ 赢 21% | **方向反转** |

**关键教训**：DiT 实验必须使用 head-to-head（同 run）比较。

## 4. 统一理论框架

### 4.1 Cosh 族足够 — 不需要新族

实验否定了 Power-Shift，验证了 Cosh 族在 DiT 上的有效性。统一叙事：

> EVQ-Cosh 是 RoPE 频率分配的通用闭式解。
> 唯一的架构依赖参数是 τ*：
> - AR causal: τ*_AR = K/√T（激进低频增强）
> - DiT bidirectional: τ*_DiT ≈ 0.53 × K/√T（温和低频增强）

### 4.2 DiT 缩放因子 γ 的物理意义

γ = τ*_DiT / τ*_AR ≈ 0.53

- AR 只需优化目标 A（传播距离）→ 全力低频增强
- DiT 同时优化目标 A + B（传播 + 指纹）→ 折中

γ ≈ 0.5 意味着 DiT 在两个目标之间大致各分一半权重。

### 4.3 离散相变假说

τ=1.2 到 τ=1.5 的急剧跳变可能源于 base=10000 在短时间序列（T_train=32）上产生的"死通道"效应：

- base=10000, K_t=16: θ_k = 1/10000^(2k/32) for k=0..15
- 高 k 的通道频率极低，θ_k × Δ ≈ 0 for all relevant Δ∈[1,32]
- 这些"死通道"不提供任何位置区分力
- 在特定 τ 阈值处，EVQ 将足够多的频率资源从死通道转移到有效区间，触发性能跳变

**待验证实验**：base_t sweep {100, 256, 500, 1000, 5000, 10000} 预计在小 base_t 下消除死通道，使 τ 曲线变得平滑。
