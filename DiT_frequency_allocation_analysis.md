# DiT 频率分配理论分析

## 1. 问题背景

EVQ-Cosh 在 AR 模型上已证明有效：通过将部分高频资源挪向低频端，牺牲短距离精度换取长距离外推能力。τ* = K/√T 是 AR 的最优浓度参数。

但在 DiT (bidirectional attention) 的 129.6M 实验中，τ=2.83 的 EVQ 被 GEO 大幅击败（denoising precision 差 45-71%）。38.8M 模型上 EVQ 仍然胜出。

核心问题：**DiT temporal 维度的最优频率分配是什么？**

---

## 2. AR vs DiT：RoPE 频率的不同角色

### 2.1 AR（Causal Attention）

RoPE 频率服务于**信息传播**：

- Causal mask 限制了注意力方向，信息只能从左到右传播
- 低频分量让注意力在长距离上衰减更慢，支撑远距离信息传递
- EVQ 的低频增强直接提升了传播距离和外推能力
- 中频损失可以接受——causal mask 本身就限制了注意力范围，不需要中频做位置区分

优化目标：最大化注意力核 A(Δ) = Σ_k cos(θ_k·Δ) 在距离 L 处的幅值

最优解：τ*_AR = K/√T（大τ，激进低频增强）

### 2.2 DiT（Bidirectional Attention）

RoPE 频率同时服务于**两个目标**：

**目标 A — 频谱匹配（Video-RoPE 的 insight）**

视频时间维度变化慢，相邻帧高度相关。temporal power spectrum 集中在低频。RoPE 需要更多低频分量来自然编码这种慢变化，让 attention 更容易学到正确的时间相关性。

→ 需要低频增强（方向与 AR 相同）

**目标 B — 位置指纹（Bidirectional 特有需求）**

没有 causal mask，所有帧互相可见。RoPE 是模型区分不同帧的唯一线索。位置混淆度定义为：

$$C(\Delta) = \frac{1}{K}\sum_k \cos(\theta_k \cdot \Delta)$$

理想情况 C(Δ) ≈ 0 对所有 Δ ≥ 1。中频段是区分"中等距离帧对"（Δ=5-20）的关键——高频在这些距离上振荡抵消，低频变化太慢贡献不大，中频恰好在这个尺度上提供区分力。

→ 中频不能被掏空

**关键矛盾：AR 不在乎中频（causal mask 代劳），DiT 必须保住中频。**

τ=2.83 对 DiT 的失败原因：为了低频增强而过度抽空中频，破坏了 fingerprinting。

---

## 3. 为什么不是简单地"GEO 最优"

之前的错误分析：既然中频重要，那 GEO（均匀分布）就是最优。

问题：这忽略了目标 A。Video-RoPE 已经证明 temporal 需要更多低频。GEO 在低频端不够。纯 GEO 的 fingerprint 最好，但频谱匹配最差——两个目标需要平衡。

**正确的分析：DiT temporal 需要"温和的"低频增强——比 GEO 多一些低频，但远不到 AR 的程度。**

---

## 4. 两条设计路线

### 路线 A：EVQ-Cosh 族内调参

最简单的方案：用同一个 Cosh 族，但 τ 更小。

理论预测：

$$\tau^*_{DiT} \approx \gamma \cdot \frac{K_t}{\sqrt{T_{train}}}, \quad \gamma \in [0.3, 0.5]$$

对于 K_t=16, T_train=32：

- τ*_AR = 16/√32 ≈ 2.83
- τ*_DiT ≈ 0.85 ~ 1.4

γ 的物理意义：DiT 的低频需求来自"频谱匹配"（温和），AR 的低频需求来自"信息传播"（强烈）。匹配需求约为传播需求的 1/3 到 1/2。

**优点**：框架统一，论文叙事简洁——同一个 EVQ-Cosh，不同架构不同 τ*

**缺点**：Cosh 分布的形状是为 AR 推导的。它在低频增强的同时，对中频的影响方式可能不是 DiT 最优的。

### 路线 B：DiT 专用频率族（Power-Shift）

设计目标：**只增强低频，几乎不动高频，轻微压缩中频**

提出单参数族：

$$\phi_k(\alpha) = 1 - (1 - u_k)^{1+\alpha}, \quad \alpha \geq 0, \quad u_k = \frac{k}{K-1}$$

性质分析：

- α = 0：φ_k = u_k，退化为 GEO
- α > 0：所有频率向低频端偏移，**偏移量与原始频率的高低成正比**

各频段的偏移行为：

| 频段 | u_k 范围 | 偏移量 ≈ | 影响程度 |
|------|---------|----------|---------|
| 高频 | 0 ~ 0.2 | α·u_k² (二阶小量) | 几乎不变 |
| 中频 | 0.3 ~ 0.7 | 中等 | 轻微压缩 |
| 低频 | 0.8 ~ 1.0 | 显著 | 大幅增强 |

数值对比（K=16）：

| slot | GEO | Cosh τ=2.83 | Power-Shift α=0.5 |
|------|-----|-------------|-------------------|
| k=1 (高频) | 0.067 | ~0.02 | 0.069 |
| k=4 (中高频) | 0.267 | ~0.10 | 0.295 |
| k=8 (中频) | 0.533 | ~0.22 | 0.596 |
| k=12 (中低频) | 0.800 | ~0.58 | 0.891 |
| k=14 (低频) | 0.933 | ~0.98 | 0.983 |

关键对比：

- **Cosh τ=2.83**：高频剧烈压缩（0.067→0.02），中频被大幅抽走（0.533→0.22）
- **Power-Shift α=0.5**：高频几乎不动（0.067→0.069），中频轻微移动（0.533→0.596），低频显著增强（0.933→0.983）

**优点**：形状专门为 DiT 设计，fingerprint 保护更好

**缺点**：引入新族，需要额外理论论证

---

## 5. 理论框架：统一叙事

论文可以建立的统一框架：

> RoPE 频率分配的最优解取决于注意力机制的类型。我们将频率设计分解为两个独立维度：
>
> 1. **分布形状**（shape）：频率 slot 在频谱上的分布方式
> 2. **分布位置**（location）：整体频率范围的偏移（即 base 参数）
>
> 对于分布形状：
> - AR causal attention → EVQ-Cosh(τ*)，τ* = K/√T（传播优化）
> - DiT bidirectional attention → EVQ-Cosh(τ*_DiT) 或 Power-Shift(α*)（频谱匹配 + 指纹优化）
>
> 两者都是 quantile-based frequency design 的特例，区别仅在优化目标不同。

这把一个"EVQ 在 DiT 上失败"的 negative result 变成了"**框架足够 general，能为不同架构推导不同最优解**"的 positive result。从一个 contribution 变成两个。

### 6.1 当前状态

### 6.2 完整实验矩阵

**Cosh 族 τ sweep**（验证路线 A）：

| τ | 预期效果 |
|---|---------|
| 0.0 | GEO baseline |
| 0.3 | 极轻微低频增强 |

**Power-Shift 族 α sweep**（验证路线 B）：

| α | 预期效果 |
|---|---------|
| 0.0 | GEO baseline |
| 0.25 | 轻微低频增强，高/中频几乎不变 |
| 0.5 | 中等低频增强，中频轻微压缩 |
| 0.75 | 较强低频增强 |
| 1.0 | 强低频增强 |

### 6.3 评估指标

- Denoising precision（主指标）：train region + extrapolation region + far-extrapolation (with YaRN)
- 每个配置只需替换 inv_freq 后 evaluate（如果不需重新训练则成本极低）

### 6.4 预期结果

1. Cosh 族：最优 τ 在 0.8-1.5 区间，显著优于 τ=0（GEO）和 τ=2.83（AR 最优）
2. Power-Shift 族：最优 α 在 0.3-0.75 区间
3. Power-Shift 最优点 ≥ Cosh 最优点（因为形状更匹配 DiT 需求）
4. 如果预期 3 成立，论文有两个 contribution；如果 Cosh 同样好，论文叙事更简洁

## 7. 129.6M 实验结果的重新解读

此前的解读："EVQ 不适合 DiT"

修正后的解读：

- **38.8M EVQ 赢**：小模型 noise floor 高，频率分配差异影响小。EVQ 的低频增强方向本身是对的，即使 τ=2.83 过度损害了中频，38.8M 也感知不到
- **129.6M GEO 赢**：大模型能充分利用频率分配。τ=2.83 过度抽空中频 → fingerprint 崩坏 → denoising 恶化。GEO 胜出不是因为"不需要低频增强"，而是因为"τ=2.83 的低频增强副作用太大"
- **预测**：129.6M 在 τ≈1.0 或 α≈0.5 时应该**同时优于 GEO 和 EVQ(2.83)**

如果这个预测被验证，就是最强的实验证据。

## 8. 待思考的开放问题

1. **α 的理论最优值能否从视频数据的时间相关长度推导？** 如果 temporal autocorrelation 衰减为 R(Δ) ∝ exp(-Δ/T_corr)，α* 应该是 T_corr 的函数。

2. **Power-Shift 族和 Cosh 族能否统一为一个两参数族？** 比如 φ_k(τ, α) 同时控制对称/非对称低频增强。AR 用 (τ*, 0)，DiT 用 (0, α*)。

3. **如果 Cosh(τ≈1.0) 已经和 Power-Shift(α≈0.5) 效果相当，是否值得引入新族？** 如果 Cosh 在小 τ 下形状和 Power-Shift 很接近，那 Occam's razor 倾向于只用 Cosh + 不同 τ。

4. **base_t 维度是否需要同时调？** Video-RoPE 的 insight 是 base_t > base_s。当前实验 base_t = base_s = 10000。如果资源允许，2D sweep (τ, base_t) 能给出更完整的图景。
