# Low-Priority Experiment Archive

This directory stores smoke tests, probe runs, and debugging artifacts moved out of the main `results/` root.

## Why it exists

- Keep the primary `results/` view focused on publishable or advisor-facing experiments.
- Preserve all historical artifacts for traceability.

## What belongs here

- Batch-size/memory probe runs
- Smoke checks
- Debug diagnostics
- Temporary migration leftovers
