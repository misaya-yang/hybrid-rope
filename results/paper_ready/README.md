# Paper-Ready Evidence Bundle

This folder is a lightweight advisor-facing mirror of key experiments.

## Inclusion rule

- Only summary/evidence files are copied here:
  - `.json`
  - `.md`
  - `.txt`
  - `.csv`
  - `.pdf`
  - `.png`
  - `.log`

No model weights are stored in this folder.

## Suggested usage

- Use this directory as the first source when building slides and report tables.
- If full run traces are needed, go back to the corresponding source folder under `results/`.
