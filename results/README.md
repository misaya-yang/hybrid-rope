# results 目录说明

本目录保存“聚合后的关键结果快照”，便于快速查阅。

## 子目录

- `unified_search/`：统一搜索（A split）结果
- `unified_search_3cfg_3seed/`：3 配置 × 3 seed 稳健性结果
- `50m_yarn_compare_v2/`：Hybrid vs Geo vs Geo+YaRN 对比结果
- `350m_final/`：350M final 对照结果
- `figures/`：核心图表输出
