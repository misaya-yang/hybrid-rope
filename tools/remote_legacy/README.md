# Remote Legacy Scripts

This folder contains historical one-off remote operation scripts that were previously in project root.

## Included categories

- `upload_*.py`
- `download_*.py`
- legacy root `run_*.py` launchers
- remote helper utilities (`seetacloud_plink.py`, `remote_*`)

## Notes

- These scripts are kept for backward traceability.
- New remote automation should be added to `scripts/` (experiment logic) or `tools/` (ops tooling), not root.
