# Paper Draft Workspace

This directory contains only submission-facing paper assets.

## Read Order

1. `mainstory.md`
2. `CORE_THEORY.md`
3. `figs/README.md`
4. `NEURIPS_SUBMISSION_PLAN.md`
5. `submission/main.tex`

## Main Files

| File | Role |
|------|------|
| `mainstory.md` | one-page spotlight-oriented narrative spine |
| `CORE_THEORY.md` | primary theory + evidence source of truth |
| `SECONDARY_THEORY.md` | appendix-level / secondary claims |
| `PAPER_PLAN_V9.md` | historical planning notes |
| `NEURIPS_SUBMISSION_PLAN.md` | current anonymous-submission constraints and page plan |
| `LATEX_SNIPPETS.md` | reusable prose / math snippets |
| `PAPER_ERROR_CORRECTIONS.md` | known wording and claim corrections |

## Figures

`figs/` is the canonical home for paper-facing figures and the

## Submission Source

`submission/` contains the anonymous NeurIPS draft:

- `main.tex`
- `sections/`
- `tables/`
- `appendix/`
- `refs/references.bib`

Only the anonymous paper source should live there. Build artifacts are ignored.
