# 论文故事线与主张

> 最后更新：2026-02-22（加入公平 8B 协议与 anchored_hybrid 方法）

## 1. 论文主叙事

### 一句话摘要

> 我们证明频谱形状设计是改善 RoPE 长程外推的有效维度，提出高频锚定的 anchored_hybrid 方法，在从零训练和公平 8B LoRA 微调上验证有效性，并系统揭示失效边界与修复路径。

### 叙事三段论

1. **问题**：标准几何 RoPE 在超长区间的相位冲突导致远距离区分能力崩溃。
2. **方法**：频率资源重分配 — 高频区保留短程精度（锚定），低频区提升长程可分辨性（平滑混合）。
3. **证据**：多规模从零训练 + 公平 8B LoRA + Phase Collision 机理指标 + 失败模式与修复。

## 2. 核心主张与置信度

### 主张 A — 高置信 ✅

- 崩溃比改善从 22x → 1.08x

## 3. 贡献点表述

1. **频谱形状视角**：提出频率分布形状作为 RoPE 长程稳定性的独立设计维度
2. **Anchored Hybrid 方法**：高频锚定 + 低频平滑混合，兼顾短程精度与长程外推
3. **跨规模实证**：50M→350M 从零训练 + 8B LoRA 公平协议
4. **失败模式图谱**：高频污染、模板遗忘、注入失效等失效模式及修复路径
