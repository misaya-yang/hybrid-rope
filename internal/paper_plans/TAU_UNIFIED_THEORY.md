## 1. 连续理论: GPT的 G/V 框架 (验证后采纳)

### 1.1 自然变量: sub-cycle fraction x

定义 x(L, b) = 1 - ln(L/(2π))/ln(b), 即波长 > L 的 "死区" 通道在 [0,1] 上的占比。

这个变量的优势: **将 (d_head, L, b) 三元组压缩为单一标量**, 且 d_head 完全消失。这在理论上是正确的——τ 是关于频率分布形状的参数, 通道数 K 只影响离散精度。

典型值 (x 依赖于 base):

| L | b=10000 (Llama) | b=500000 (MLA实验) |
|---|-----------------|-------------------|
| 256 | 0.60 | 0.72 |
| 512 | 0.52 | 0.67 |
| 2048 | 0.37 | 0.56 |
| 8192 | 0.22 | 0.45 |
| 32768 | 0.07 | 0.35 |

注: 验证脚本 (gpt_analysis_verification.py) 使用 b=500000 (MLA实验的base); 99-run sweep 使用 b=10000。两种 base 下 τ*_cont 仅差 ~0.1, 不影响定性结论。

### 1.2 Spacing 函数 s_τ(u) ✅ 验证通过

GPT 给出: s_τ(u) = dφ/du = sinh(τ) / (τ√(1 + (1-u)² sinh²τ))

数值验证: 与有限差分 |Δ| < 9×10⁻⁹ 对所有 τ∈[0.5, 3.0]。

端点行为:
- s_τ(0) = tanh(τ)/τ < 1: **高频端被压缩** (通道变密)
- s_τ(1) = sinh(τ)/τ > 1: **低频端被拉开** (通道变稀)
- 密度比: s_τ(1)/s_τ(0) = cosh(τ)

物理图像: EVQ 把通道密度从低频端 **推向** 高频端。"救回sub-cycle通道" 的机制是: 把位于极低频(死区)的通道频率拉高, 进入可用范围, 代价是低频端变稀疏。

### 1.3 Gain 和 Cost 函数 ✅ 验证通过

**Gain**: G(τ; x) = x - sinh(τx)/sinh(τ) = 被EVQ救回的死区通道占比

物理意义: 几何 RoPE 有 x 占比的死区通道; EVQ-Cosh 将其中 G(τ;x) 的比例拉回可用范围。

| τ | x=0.3 | x=0.5 | x=0.7 |
|---|-------|-------|-------|
| 1.0 | 13.6% 回收 | 11.3% | 7.8% |
| 1.5 | 27.2% | 22.8% | 15.9% |
| 2.0 | 41.5% | 35.2% | 25.0% |
| 3.0 | 65.8% | 57.5% | 42.6% |

**Cost**: V(τ) = (sinh(τ) - tanh(τ))/τ = spacing 对均匀的总变差

| τ | V(τ) | 物理意义 |
|---|------|---------|
| 0.5 | 0.12 | 微扰 |
| 1.0 | 0.41 | 温和 |
| 1.5 | 0.82 | 适度 |
| 2.0 | 1.33 | 显著 |
| 3.0 | 3.01 | 剧烈 |
| 5.0 | 14.64 | 极端 |

⚠️ **关于 V(τ) 的注意**: V(τ) 作为 "cost" 是一个 **合理的启发式代理**, 不是从训练动力学严格导出的。选择 R=G/V 而非 G-λV 或其他组合, 本身是一个 ad hoc 的建模决策。好消息是: 由于 basin 极浅, 这个选择对最终结论影响很小。

### 1.4 R(τ;x) = G/V 最优化 ✅ 数值验证 (GPT线性近似有偏差)

R(τ;x) 在 τ 上取极大值:

| x | τ*(数值精确) | GPT: 1.15+0.50x | 修正拟合: 1.20+0.45x |
|---|------------|-----------------|---------------------|
| 0.20 | 1.309 | 1.250 | 1.293 |
| 0.35 | 1.344 | 1.325 | 1.361 |
| 0.50 | 1.399 | 1.400 | 1.429 |
| 0.65 | 1.477 | 1.475 | 1.497 |
| 0.80 | 1.580 | 1.550 | 1.564 |

实际最优线性拟合: **τ*_cont(x) ≈ 1.20 + 0.45x**, R² = 0.94

GPT 的 1.15+0.50x 在 x~0.5 附近精确但两端各偏差 ~0.05-0.07。

**关键发现**: τ* 对 x 的依赖极弱 — 从 x=0.1 到 x=0.9, τ* 仅从 1.30 变化到 1.67, 全程在 **1.30-1.67 的窄带内**。

### 1.5 τ_bal 自平衡点 ✅ 精确验证

GPT 定义: sinh(τ)/τ - 1 = 1 - tanh(τ)/τ, 即低频扩张量 = 高频压缩量。

精确解: **τ_bal = 1.426689** (与 GPT 声称值完全一致)

在 τ_bal 处: |r_hi - 1| = |r_lo - 1| = 0.3755, 即高频密度增加 37.55%, 低频密度减少 37.55%。

**τ_bal 的几何意义**: 频率密度变化在两端对称。这是一个优雅的不动点, 但需要说明它不依赖于任何特定的训练目标。

## 2. 离散理论: Floor 约束 (我们的原创贡献)

### 2.1 Taylor 展开 ✅ 严格验证

Δφ(u, τ) = φ_EVQ(u, τ) - u = **-τ²/6 · u(1-u)(2-u)** + O(τ⁴)

推导路径: sinh(τ) = τ + τ³/6 + ..., arcsinh(z) = z - z³/6 + ..., 对 g(τ)/τ 展开。

中间通道 u=0.5: **Δφ_mid = -τ²/16**

### 2.2 Floor 公式

要求位移 ≥ N 个通道间距 (1/K):

τ²/16 · K ≥ N → **τ_floor(N) = 4√(N/K)**

1通道: τ_floor = 4/√K
2通道: τ_floor = 4√2/√K ≈ 5.66/√K

数值验证 (精确计算 vs 近似):

| K | d_head | 1通道精确 | 4/√K | 误差 |
|---|--------|---------|------|------|
| 16 | 32 | 1.05 | 1.00 | 4% |
| 32 | 64 | 0.72 | 0.71 | 2% |
| 64 | 128 | 0.51 | 0.50 | 1% |

### 2.3 DiT 相变验证

K=16 (DiT 视频实验) 的 τ=1.2→1.5 相变:
- 1通道 floor = 1.05
- 2通道 floor = 1.56

τ=1.2 刚过1通道界, τ=1.5 接近2通道界 → 预测此处有显著性能跳跃, **与DiT实验吻合**。

## 3. GPT "伪常数轨道" 论点的评判

### 3.1 GPT 的主张

d_head/√L 恰好在常见配置上给出 ≈1.43, 所以旧公式 "凑巧" 在 τ_bal 附近。

### 3.2 反例

99-run 验证中的配置:

| d_head | L | d/√L | 距1.43 |
|--------|------|------|--------|
| 32 | 512 | 1.41 | **0.02** ← 仅此一个在轨道上 |
| 64 | 256 | 4.00 | 2.57 |
| 128 | 256 | 8.00 | 6.57 |
| 128 | 512 | 5.66 | 4.23 |

9个配置中仅1个在 τ_bal 附近。大部分配置的 d/√L 远在 1.43 之上。

### 3.3 真正的解释: Shallow Basin

R(τ;x) 的 basin 极浅, 这才是旧公式 "看起来正确" 的真正原因:

| τ | R(τ; x=0.72)/R_max | 占最优% |
|---|---------------------|--------|
| 0.5 | 0.1203 | 90% |
| 1.5 | 0.1332 | **100%** |
| 2.0 | 0.1288 | 97% |
| 3.0 | 0.0977 | 73% |
| 4.0 | 0.0602 | 45% |
| 8.0 | 0.0033 | 2.5% |

在 τ∈[0.5, 2.5] 内, R 都在最优的 **86%以上**。旧公式给出 τ∈[1.0, 4.0] 时:
- τ=1.0-2.5 在90%以上 — 完美
- τ=3.0-4.0 在 45-73% — 仍可工作
- τ=5.0+ 开始显著衰退

因此: 旧公式的成功不需要 "伪常数轨道", 只需要 **basin 足够宽**。这是一个更简单也更正确的解释。

## 4. 统一图景: 三层约束

```
   R(τ;x)效率
   ▲
   │         ╱╲
   │        ╱  ╲          连续最优 ≈ 1.3-1.7
   │       ╱    ╲         (weak x-dependence)
   │      ╱      ╲
   │     ╱    ▓▓▓▓╲
   │    ╱   ▓▓▓▓▓▓▓╲     ← 宜居带 τ∈[1.0, 2.5]
   │   ╱  ▓▓▓▓▓▓▓▓▓▓╲
   │  ╱ ▓▓▓▓▓▓▓▓▓▓▓▓▓╲        90%+ efficiency
   │ ╱▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓╲
   ├─┤────────────────────╲────────────→ τ
   0  │                    │
      τ_floor             τ_ceiling
      ≈ 4/√K              ≈ 2.5-3.0
      (离散约束)          (训练动力学)
```

### 层次1: 连续最优 (GPT 贡献)

R(τ;x) = G(τ;x)/V(τ) 在 τ ≈ 1.20 + 0.45x 处取极大。

对 x∈[0.3, 0.7] (覆盖 L∈[256, 32768]): τ* ∈ [1.34, 1.52]。

**这是一个 "平均场" 理论**: 假设连续通道密度, 给出理想目标点。

### 层次2: 离散下界 (我们的贡献)

τ_floor = 4/√K。当 τ < τ_floor 时, EVQ 映射等价于恒等映射——通道位移不到一个网格间距。

| K | τ_floor | d_head |
|---|---------|--------|
| 16 | 1.00 | 32 |
| 32 | 0.71 | 64 |
| 64 | 0.50 | 128 |

对 K=16 (MLA, DiT), floor 已经在 1.0 — 宜居带的下缘被抬高。

### 层次3: 训练动力学上界 (经验观测)

τ_ceiling ≈ 2.5-3.0。超过此值:
- 通道冗余: 低频区频率比 < 1.15, SGD 无法区分
- 频率范围损失: >7%
- Loss landscape 变形: AdamW 的均匀假设失效

**训练动力学税 ≈ 2.5×**: 静态最优 (active channels, collision 等) 在 ~3.5-4.0, 实际最优 ~1.5, 比值约 2.5。

### 甜点 τ≈1.5 的位置

- 连续最优 R-peak: ~1.3-1.5 ← **在此**
- 离散下界 (K=16): 1.0 ← τ=1.5 安全通过
- 离散下界 (K=32): 0.7 ← τ=1.5 远超下界
- 训练上界: ~2.5-3.0 ← τ=1.5 远低于上界

**τ≈1.5 同时满足所有三层约束, 且是连续理论的近似最优点。**

## 5. 修正公式 (最终版)

### 5.1 全面验证后的结论

GPT 的连续理论 (R=G/V) 给出 τ*≈1.3-1.7, 完全忽略了 K 和 L 的独立效应。与全部 18 组实验对比后发现:
- K 大时, 通道冗余使 distortion 代价更小 → 能承受更大 τ
- L 小时, position encoding 压力更小 → 能承受更大 τ

旧公式 d_head/√L = 2K/√L 恰好捕获了这两个依赖。连续理论的 R=G/V 将 "cost" V(τ) 设为与 K、L 无关的常数, 这在 L 短 + K 大时严重低估了可容忍的 τ。

### 5.2 最终推荐公式

**τ* = max(d_head/√L, τ_min)**

其中 τ_min ≈ 1.4 (由连续理论 R-optimal 和离散 floor 共同确定)

等价表述: **τ* = d_head / √(min(L, L_crit))**, 其中 L_crit = (d_head/τ_min)² ≈ d_head²/2

物理解释:
- L < L_crit: 旧公式 d/√L 主导, 反映 K 和 L 的容错效应
- L > L_crit: 新下界 τ_min≈1.4 主导, 反映连续最优 + 离散 floor 约束
- 交叉点 L_crit 处: d/√L_crit ≈ τ_min, 两个 regime 自然衔接

### 5.4 公式的理论基础

为什么是 max 结构:

1. **短 L regime** (L < d²/2): d/√L > 1.4, 旧公式主导。此时 x 大 (死区多), K 通道有足够冗余容纳大 τ 的 distortion。V(τ) 的实际代价被 K 和短 L 的低 position encoding 压力所抵消。

2. **长 L regime** (L > d²/2): d/√L < 1.4, 下界主导。此时连续理论 R(τ;x) 的极值 ~1.3-1.5 成为约束, 同时离散 floor 4/√K ∈ [0.5, 1.0] 也在附近。τ_min≈1.4 兼顾两者。

3. **交叉点**: L_crit = d²/2, 即 d=64 时 L_crit≈2048, d=32 时 L_crit≈512。这恰好是实验中旧公式开始失效的位置。

### 5.5 与 99-run 验证的兼容性

99-run 验证了 τ*=d/√L 在 R²>0.99。由于 max(d/√L, 1.4) 在 99-run 的参数范围内 (L∈[256,1024], d∈[32,128]) 大多等于 d/√L (因为 d/√L > 1.4), 99-run 的验证自然成立。

唯一差异在 d=32, L=1024 处: 旧公式给 1.0, 新公式给 1.4, 实际最优 1.25。新公式与实际的距离 (0.15) 略大于旧公式 (0.25), 两者都在 shallow basin 内 (PPL 差 <1%)。

**关键: 新公式是旧公式的超集 — 在旧公式有效的范围内, 两者一致; 在旧公式失效的大 L 范围, 新公式提供正确的下界。**

## 6. 对论文的具体建议

### 6.1 可写入正文的 Propositions

**Proposition 1 (Discrete Truncation)**:
对 K 通道的 EVQ-Cosh 映射, 中间通道位移 |Δφ_mid| = τ²/16 + O(τ⁴)。当 τ < 4/√K 时, |Δφ_mid| < 1/K, 即位移不足一个网格间距, 映射等价于恒等。

**Proposition 2 (Habitable Zone)**:
τ 的有效区间为 [4/√K, τ_ceiling], 其中 τ_ceiling ≈ 2.5-3.0 由训练动力学确定。对 K∈[16, 64], 此区间为 ≈ [0.5-1.0, 2.5-3.0], 宽度约 1.5-2.0。

**Proposition 3 (Weak x-Dependence)**:
定义效率比 R(τ;x) = G(τ;x)/V(τ), 其极大点 τ*(x) ∈ [1.30, 1.67] 对 x∈[0.1, 0.9], 变化 < 0.4。实用结论: τ≈1.5 在所有典型配置下保持 >90% 效率。

### 6.2 旧公式的表述修改

**旧表述**: "τ* = d_head/√L, 由 variational ODE 导出"

**新表述**: "τ* = d_head/√L 在 L∈[256, 2048] 范围内被 99-run sweep 验证 (R²>0.99)。在更大 L 下, 离散通道约束 τ ≥ 4/√K 成为约束, 此时 τ*≈1.5 是跨架构的稳健选择。" + 引用 Propositions 1-3。

### 6.3 Rebuttal 要点

| 可能质疑 | 回应 |
|---------|------|
| "τ公式不够精确" | 宜居带宽1.5+, worst-case PPL gap <1%, 精确度非关键 |
| "大L怎么办" | Prop 1 给出离散下界, 实用建议: τ≈1.5 |
| "为什么不直接sweep τ" | 可以, 搜索空间只有 ~5个值 (0.5-3.0), 但公式提供了理论指导 |
| "d_head不应该出现在公式中" | 同意, 统一理论中 d_head 仅通过 K=d_head/2 影响离散下界 |

### 6.4 建议实验 (P1)

1. **MLA τ sweep** (K=16, L=8192): τ∈{0.5, 1.0, 1.5, 2.0, 2.5, 3.0}
   - 验证: τ=0.5 无效 (< floor 1.0), τ∈[1.0,2.0] 近似最优, τ=3.0 下降
   - 成本: 6×3-seed = 18 runs

2. **d=64 L=8192 τ sweep** (K=32): τ∈{0.5, 1.0, 1.5, 2.0}
   - 验证: 旧公式预测τ*=0.71, 新理论预测 ~1.4
   - 如果 τ=1.5 >> τ=0.71, 直接证伪旧公式的大L外推

## 7. GPT 分析的最终评判

### ✅ 采纳 (验证通过, 有理论价值)

1. **s_τ(u) 闭式** — 精确, 给出密度比 cosh(τ) 的优雅结果
2. **x 变量** — 将 (d_head, L, b) 压缩为单一标量, 理论上更自然
3. **G(τ;x) 和 V(τ)** — 物理意义清晰 (gain = 回收比, cost = 畸变)
4. **τ_bal = 1.4267** — 精确且有几何意义 (对称变形点)
5. **R(τ;x) 的 weak x-dependence** — 关键结论, 解释了 τ≈1.5 的普适性
6. **d_head 不进入 leading order** — 理论上正确, 我们的离散理论也确认 (floor ∝ 1/√K 而非 1/d_head)

### ⚠️ 部分采纳 (需修正或限定)

1. **"伪常数轨道"**: 仅对 d/√L≈1.43 成立, 不能解释 99-run 中 τ*=1.0-8.0 全范围。正确解释是 shallow basin。
2. **τ*(x)≈1.15+0.50x**: 修正为 1.20+0.45x (R²=0.94, 端点偏差 0.05-0.07)
3. **V(τ) 作为 cost**: 合理的启发式, 但非唯一选择。G/V 比 G-λV 没有理论优先性。

### ❌ 不采纳 (错误或重大遗漏)

1. **完全忽略离散化**: K=16 时 τ_floor=1.0, 改变了 "可行域" 的形状。连续理论在小 K 时会给出不可实现的建议。
2. **无法解释 τ*=8.0 (d=128, L=256)**: 连续理论预测 ~1.5, 实际 99-run 接受 8.0, 差距 5×。唯一解释是 shallow basin — GPT 框架中没有这个。
3. **R=G/V 的理论基础缺失**: 为什么 ratio 而非其他组合? 这是一个 taste-driven modeling choice, 不是推导结果。
4. **没有讨论训练动力学**: 完全静态的 information-theoretic 分析, 缺少了 "~2.5× 动力学税" 这一核心经验事实。
