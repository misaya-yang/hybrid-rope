## 问题 1: λ 的解析闭合 —— ✅ 完整解决

### 旧状态（2026-03-31）
λ 通过拟合实验 τ* 值校准；CV 11.1%；"是否为常数或简单函数" 开放。

### 新状态（2026-04）

**Q4 给出了 invariant 定义**（Appendix `sec:lambda-curvature`, Eq. `eq:lambda-true`）：
```
λ_true(τ,L,b) = ∂_θ S_χ²(ρ_√θ) / ∂_θ U_exact(ρ_√θ; L, b) |_{θ=τ²}
λ_∞ := lim_{τ→0} λ_true = u₂^exact / u₂^unit
```

**关键洞察**：λ 不是自由拟合参数，而是 **surrogate 二阶变分 vs exact 二阶变分的曲率比**。λ=1 对应完全匹配；λ_coll=1.171 = 17% surrogate 曲率低估，不是随机拟合误差。

**additional**：CV 从 11.1% 降到 0.11%（`tab:lambda-cv`，改名为 c_coll 以区分 prefactor 和 curvature multiplier）。

### 剩余开放项

- **验证 u₂^exact 积分**：GPT 给的 protocol 是"在 12 个 exact-kernel config 上数值积分 u₂^exact = ∂_{τ²} U_exact(ρ_τ; L, b)|_{τ=0}，看比值是否 = 1.17"。这是 **offline numerical 计算，<1h，无 GPU**。
- 如果跑出来 = 1.17，则 c_coll 从 "empirical calibration" 升为 "predicted ratio"，论文可以说 "we derive c ≈ 1.17 from surrogate-exact curvature mismatch on our 12 configs"。

## 问题 3: DiT 模态修正因子 —— 🔶 理论完成待测量

### 旧状态
τ*_DiT ≈ 0.53 × d/√L，post-hoc 拟合，分解为 "bidirectional × noise" 也是事后解释。

### 新状态

**A4 给出了 theorem-level 推导**：
```
τ_video / τ_causal = √(η_VP^(m) · M_c/M_b) ≈ √(η_VP^(m) / 2)
```

其中：
- **1/√2 来自 bidirectional softmax Jacobian 谱**（严格推导，不是事后 rationalize）
- **η_VP = ∫ w(t) α_t² dt** 是 VP SDE 下 signal-variance schedule 积分
- m=2 或 m=4 取决于 utility 定义（covariance-linear 还是 energy-squared）
- 我们论文的 U 是 covariance-linear → **m=2**

**对应数字**：经验 0.53 ⇔ η_VP ≈ 2 × 0.53² = 0.562

### 剩余开放项（紧急度：高，但<1h 就能解决）

- **必须做的一件事**：算 `η_VP = mean(w_t · α_t²)` over your DiT training schedule
- 位置：你之前训练 video DiT 的 diffusion schedule config 文件（可能在 `scripts/video_dit_*.py`）
- 预期：如果算出来 ≈ 0.56，A4 的 theorem 立即成立，`§6 Limitations` 里的 "rationalization not derivation" 可以改成 "derived via Eq. (56)"
- 如果偏差大（例如 η_VP = 0.3 或 0.8），则 A4 的假设被 falsify，我们把它作为 limitation 诚实写出

**无 GPU 需求**。

**GPT 的审稿指出这个证明有严重 bug**：原推导假设 δω ∝ τ 给出 Var(δℓ) ∝ τ²，但 EVQ warp 实际 δω = O(τ²)（Eq. `eq:quantile-displacement`），所以 frozen-weight mismatch 应该是 τ⁴ 阶，不是 τ²。

**我们的选择**：**改用 phenomenological 叙事**，不再声称严格推导。Appendix `sec:lora-regime` 现在写：

> "This subsection gives a phenomenological model calibrated by the rank sweep, not a first-principles derivation: the EVQ warp actually perturbs frequencies at order τ², so the τ² coupling below is an effective single-scale surrogate for the leading non-perturbative interaction with frozen pretrained Q/K blocks."

**追求严格证明的路径依然开放，但不再是必要条件**。如果未来想重启：

## 问题 6: τ* 在 L ≥ 4096 的有效性 —— ✅ 完整解决

### 旧状态
τ* 在大 L 下可能饱和，理论未定。

### 新状态

**Q1 v2 给出了完整 theorem**（Appendix `sec:leff-derivation`）：

**3 件事**：
1. **L_eff^J 定义**：Rayleigh quotient of softmax Jacobian on RoPE transport direction g
   ```
   1/L_eff^J = E[g^T J_sm(p) g] / E[||Pg||²]
   ```
2. **g_i(j) 闭式** (Eq. `eq:g-closed-form`)：从 density perturbation δρ 到 logit transport direction 的显式推导
3. **Unified exponent**：若 `L_eff^J = Θ(L_train^a)`，则 `τ* ∝ L_train^{-a/2}`；我们的 -1/2 是 a=1 的特例

**A2 又给出了 perturbative self-consistency proof**：1/L → 1/L_eff^J 不是替换，是 softmax 二阶 Taylor 的 Rayleigh 形式。

## 新出现的理论问题（2026-04）

### N1 (新) · γ(b) 的显式 Fisher-utility 推导 — ✅ 解决 (A1)
`γ(b) = (λ_F/α) η̄_F (2log b)²`。同时修正了之前 "pointwise O(1/s²) suppression" 的叙事错误：真实是 L¹/CDF O(1/log b) 级别。

### N2 (新) · Discrete→continuous transport gap — ✅ 解决 (A5)
- W_1(μ_K, μ_ρ) ≤ 1/(4Km)
- 高分辨率 distortion ~ 1/(12K²) ∫ w/ρ² dφ
- **解释 MLA (K=16) 比 MHA (K=64) 收益更大**：K⁻¹/K⁻² 定量预测 4×/16× 放大，观测 2-2.4×，directionally consistent

### N3 (新) · χ² stiffness 唯一性 — ✅ 解决 (Q3)
Load-moment 家族中只有 p=2 给 L^{-1/2}。χ² = Var_ρ(1/ρ) = "per-channel quantization load variance" 的公理化推导。

### N4 (新) · NTK destructive τ_crit — ✅ 解决 (A6)
```
τ_crit ≈ (1/φ_c) log(K / n_min)
```
Low-frequency tail mass T_τ(φ_c) = sinh(τ(1-φ_c))/sinh(τ) ~ e^{-τφ_c}。解释 Table 5 的"大 τ 下 NTK 变 destructive"。

### N5 (新 & 开放) · Waterbed 到 PPL 的连接 — 📌 开放
- 现有：`W(ρ_τ*) = O(d_head⁴/(L_eff^J)²)` 是 surrogate 不等式（Appendix `sec:waterbed-proof`）
- 缺失：W 与实测 PPL 的 link。我们说 "surrogate in-range cost is O(τ⁴)，PPL 影响也在 <1.7% 范围内，定性一致"，但没有 W → PPL 的定量连接
- **难度：高**。需要把 PPL 建模为 position-collision + content-learning 的联合泛函
- **优先级：低**。审稿人不太会追问

### N6 (新 & 开放) · 常数 C_{bc}(τ) 的显式控制 — 📌 半开放
A1 / A2 的界 `‖δρ_F‖ ≤ C_bc(τ) |γ|/|s²-τ²|` 里的边界常数 C_bc(τ) 没给显式。
- **难度：中**。需要显式求解 homogeneous 解 under natural BC
- **优先级：中**。reviewer 可能追问
