## 问题 1：Broadband 近似的算子投影误差界

```
我正在研究 Rotary Position Embedding (RoPE) 中的频率分配优化问题。核心设置如下：

精确的相位碰撞核 (phase collision kernel) 定义为：
K(φ₁, φ₂) = ∫ D(Δ) cos(b^{-φ₁}Δ) cos(b^{-φ₂}Δ) dΔ

其中 b 是 RoPE base（典型值 10⁴ ~ 5×10⁵），φ ∈ [0,1] 是归一化频率坐标，D(Δ) 是 token 距离先验分布。

我们用 broadband 近似将这个核分解为：
K(φ₁, φ₂) ≈ α·δ(φ₁-φ₂) + β·min(φ₁, φ₂)

其中 (α*, β*) 是 K 在 {αδ + βmin} 上的 Hilbert-Schmidt 最优投影。

已知结论：
- 有限 base 残差 = O(1/ln b)
- R²_mid ≈ 0.994-0.995（Table 验证）
- 但 Algorithm 1 的全矩阵 LS 拟合残差为 35-49%

问题：
1. 为什么 R²_mid 高 (>0.99) 但全矩阵残差高 (35-49%)？这两个指标衡量的是核的不同区域/不同性质吗？
2. 是否存在比 O(1/ln b) 更精确的残差上界？特别是，当 D(Δ) 是 power-law 分布 D(Δ) ∝ 1/(Δ ln L) 时，broadband 近似是否比一般情况更好（或更差）？
3. 从算子谱的角度看，{αδ + βmin} 这个二参数族捕获了精确核 K 的哪些谱成分？min(φ₁,φ₂) 是 [0,1] 上的 Green 函数（对应 -d²/dφ²），所以 αδ + βmin 本质上是 "identity + resolvent" 形式吗？如果是，能否用 resolvent perturbation theory 得到更紧的误差界？

请给出严格数学分析，包括必要的证明或证明思路。
```

## 问题 2：从 Fisher 信息到注意力效用的理论桥梁

```
在 RoPE 频率分配优化中，我用 Fisher Information 作为位置编码质量的代理度量。具体地：

对于频率 ω = b^{-φ} 的正弦位置编码，其对位置偏移 Δ 的 Fisher 信息标度为：
I(φ) ∝ ω² = b^{-2φ}

这是标准的正弦信号频率估计 Fisher 标度律（Van Trees）。

在变分框架中，我将这个 Fisher 标度作为"频率分辨力保真项"：
J[ρ] = C_interf[ρ] - μ ∫ ρ(φ) b^{-2φ} dφ + λ(∫ρ - 1)

其中 C_interf 是频率间的干涉碰撞能量，μ 是拉格朗日乘子。

问题：
1. 从 "位置偏移 Δ 的 Fisher 信息" 到 "Transformer 注意力机制中的实际效用" 之间，最严格的理论桥梁是什么？注意力权重 softmax(QK^T/√d) 中，位置编码贡献的是 cos(ωΔ) 项。Fisher 信息衡量的是对 Δ 的估计精度，但注意力机制需要的是 "在正确位置上分配高权重"——这两者在什么条件下等价？
2. 是否存在更自然的度量代替 Fisher？比如 mutual information I(position; attention_weight) 或 attention sink 的信息论分析？
3. 如果 Fisher 桥梁在某些 regime 下不成立（比如大 Δ 时 cos(ωΔ) 的高阶行为），这会如何影响最优频率分配的结论？

请给出严格分析，如果有更好的理论框架可以替代 Fisher，请详细阐述。
```

## 问题 3：cosh 族的函数逼近最优性

```
考虑如下变分问题：在 [0,1] 上求频率密度 ρ(φ) ≥ 0，∫ρ = 1，使得泛函

J[ρ] = (α/2)∫₀¹ ρ² dφ + (β/2)∫₀¹∫₀¹ ρ(φ₁)ρ(φ₂)min(φ₁,φ₂) dφ₁dφ₂ - μ∫₀¹ ρ(φ)b^{-2φ} dφ

取极值。Euler-Lagrange 方程化简后得到常系数 ODE：
ρ'' - τ²ρ = γ b^{-2φ}，其中 τ = √(β/α)

通解为 ρ*(φ) = C₁cosh(τφ) + C₂sinh(τφ) + P·b^{-2φ}。

CDF 反演后得到单参数分位数映射族：
φ_k(τ) = 1 - (1/τ) arcsinh((1-u_k) sinh τ)

问题：
1. 这个 cosh 族在什么意义上是"最优"的？具体地：给定 N 个离散频率通道，如果不限制在 cosh 族内，N 维的全自由度优化 (比如 DAPE 的 N=32 参数) 最多能比 cosh 族 (1 参数) 好多少？是否存在类似于 Kolmogorov n-width 的理论工具可以量化 cosh 族对全解空间的逼近度？
2. cosh(τ(1-φ)) 作为密度函数有一个重要性质：它在 φ=0（高频端）处密度最高，φ=1（低频端）处密度最低，且密度比 = cosh(τ)/1。这个密度比对 τ 是指数增长的。是否存在理论上界表明任何"好的"频率分配必须具有至少这种量级的高频偏置？
3. 如果将常系数 ODE 推广为变系数（即 α(φ), β(φ) 随 φ 变化），解不再是 cosh 而是某种超几何/Bessel 函数。有没有办法严格论证：在实际 RoPE 参数范围内（b = 10⁴~10⁶, N = 32~128），变系数校正相对于常系数 cosh 解的偏差有界且小？

请从函数逼近论和微分方程理论的角度给出分析。
```

## 问题 4：Waterbed 不等式的紧性与可达条件

```
在 RoPE 频率分配问题中，我们证明了一个 Waterbed 不等式：

∫₀¹ ln E(φ) dφ ≥ ln b - ln c

其中 E(φ) 是频率 φ 处的局部误差代理（相位碰撞期望），与 Fisher 信息的关系为 E(φ) ≥ 1/I(φ)，I(φ) = cρ(φ)b^{-2φ}。

证明使用 Jensen 不等式：∫ ln E ≥ ln ∫ E ≥ ln(b/c)。

实验观察：
- 在 128-token from-scratch 训练中，增大 τ 持续降低外推 PPL 且不恶化训练窗口内 PPL（即"无 waterbed"现象）
- 在下游任务中，retrieval 类任务提升而 multi-hop reasoning 下降（waterbed 出现）

问题：
1. Jensen 不等式的等号条件是 ln E(φ) = const，即 E 在频率轴上均匀。什么样的频率分配 ρ 能使 E 尽可能均匀？geometric 是否在某种意义上是使 E 最均匀的分配？
2. 为什么在 PPL 指标上没有观察到 waterbed（τ 从 0 到 5 所有值 PPL@128 几乎不变），而在下游任务上出现了？是否因为 PPL 是 token-level 平均（对 φ 的 aggregation 方式不同于 task-level 评估），使得 waterbed 效应被平均掉了？
3. 能否将 waterbed 不等式推广为更精细的形式？比如，对于给定的 τ，是否能推导出 ∫₀¹ (E(φ) - E_geo(φ))² dφ 的上下界，从而量化 EVQ 相对于 geometric 的 trade-off 幅度？
4. 是否存在类比：waterbed 不等式和信号处理中的 uncertainty principle（时频不确定性原理）有什么本质联系？RoPE 的频率分配和 Gabor 变换的窗函数选择是否存在数学同构？

请给出严格的数学推导或证明思路。
```

## 问题 5：最优 τ 随训练长度递减的理论解释

```
在 RoPE 频率优化中，我发现了一个重要的实验规律：

最优超参数 τ*（控制 cosh 频率分配的"偏置强度"）随训练序列长度 L_train 单调递减：
- L_train = 128 tokens: τ* > 5.0（单调下降，未找到 peak）
- L_train = 1024 tokens: τ* ≈ 2.0
- L_train = 2048 tokens: τ* ≈ 1.5

同时 τ* 与模型大小无关（50M 和 125M 参数的模型给出相同的 τ* 曲线）。

背景：
- τ 控制的是频率密度从均匀 (geometric, τ=0) 到高频偏置 (τ→∞) 的程度
- 模型有 d/2 个频率通道（head_dim=64 → 32 通道），这是硬约束
- 模型的 Q/K 投影矩阵也在同一个 32 维频率空间中做"软分配"

问题：
1. 能否从变分框架出发，给出 τ*(L_train) 的理论预测？具体地：当训练长度增加时，距离先验 D(Δ) 的有效支撑从 [1, L_train] 扩大，这如何改变最优的 (α*, β*) 投影系数，进而改变 τ* = √(β*/α*)？
2. τ* 与模型大小无关这一现象，是否暗示 τ* 完全由位置编码的几何结构（RoPE 频率空间的性质）决定，而与模型容量（能在 Q/K 空间做多少"软分配"）无关？如果是，能否严格证明这一点？
3. 是否存在一个 scaling law 的形式：τ*(L) = f(L) 其中 f 是某个可解析表达的函数（比如 τ* ∝ 1/ln L 或 τ* ∝ L^{-α}）？如果变分框架能给出 f 的具体形式，这本身就是一个有意义的理论贡献。
4. 从信息论角度：训练长度 L_train 越长，模型通过梯度下降"看到"的位置对越多，Q/K 矩阵能学到的位置信息越充分。τ* 的递减是否可以理解为"随着模型学到更多位置信息，位置编码需要做的'预分配'越少"？如何形式化这个直觉？

请给出理论推导或至少给出有明确数学形式的猜想 (conjecture)。
```
