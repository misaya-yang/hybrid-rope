## 1. 已严格验证（可当作定理写）

### 1.1 精确核的 Ci 闭式表达

在 power-law 截断先验
`D(Δ)= (1/(Δ ln L)) 1_{[1,L]}` 下，精确核
`K(φ1,φ2)=∫ D(Δ) cos(b^{-φ1}Δ)cos(b^{-φ2}Δ) dΔ`
可写成

`K = (1/(2 ln L))[ Ci(ω_- L)-Ci(ω_-) + Ci(ω_+ L)-Ci(ω_+) ]`,
其中 `ω_-=|b^{-φ1}-b^{-φ2}|, ω_+=b^{-φ1}+b^{-φ2}`。

要点：
1. 该表达式在对角线 `ω_-=0` 处用极限解释即可（`Ci(aL)-Ci(a)→ln L`）。
2. 该式只是一种等价写法，不涉及任何近似。

### 1.2 `min(φ,ψ)` 是 `A=-d²/dφ²` 的 Green 函数（混合边界）

令 `A=-d²/dφ²` 定义在 `[0,1]` 上，并取边界条件 `g(0)=0, g'(1)=0`，则
`A^{-1}` 的核为 `min(φ,ψ)`。

严格核等式（分布意义）是：
`-∂²_φ min(φ,ψ) = δ(φ-ψ)`。

### 1.3 Broadband 泛函的 Euler–Lagrange → ODE（含边界条件）

对 broadband 泛函

`J[ρ] = (α/2)∫ρ² + (β/2)∬ ρ(φ1)ρ(φ2) min(φ1,φ2) - μ_F ∫ ρ(φ) b^{-2φ} + λ(∫ρ-1)`

一阶变分得到积分形式的一阶条件。对 φ 方向微分两次，可严格得到：

1. **边界条件**：`ρ'(1)=-(2 μ_F ln b/α) b^{-2}`。
2. **ODE**：`ρ''(φ) - τ² ρ(φ) = γ_F b^{-2φ}`，其中 `τ = sqrt(β/α)`，`γ_F = μ_F(2 ln b)²/α`。

并且当 `τ≠2 ln b` 时通解为：

`ρ(φ)=C1 cosh(τφ) + C2 sinh(τφ) + P b^{-2φ}`,
`P = γ_F/(4(ln b)²-τ²)`。

### 1.4 纯 tether（μ_F=0）时的 cosh 密度闭式解

当 μ_F=0，ODE 退化为齐次方程且边界条件变为 `ρ'(1)=0`，归一化后唯一解为：

`ρ_τ(φ) = τ cosh(τ(1-φ))/sinh τ`。

### 1.5 EVQ：CDF 与逆 CDF（闭式 warp）与 τ→0 极限

对上述 `ρ_τ`：
1. CDF：`F_τ(φ)=1 - sinh(τ(1-φ))/sinh τ`。
2. 逆 CDF：`φ(u;τ)=1-(1/τ) asinh((1-u)sinh τ)`。
3. τ→0：`φ(u;τ)→u`（从而离散量化后退化到 geometric）。

附：局部展开（τ→0）：
`φ(u;τ)=u - (u(1-u)(2-u)/6) τ² + O(τ⁴)`。

## 2. 条件成立的严格命题（需写清前提）

### 2.1 Waterbed 不等式（Jensen + CRLB 形式）

若定义局部 Fisher 信息 `I(φ)=c ρ(φ) b^{-2φ}`，并且某个局部误差 proxy 满足
`E(φ) ≥ 1/I(φ)`，则 Jensen 给出

`∫_0^1 ln E(φ) dφ ≥ ln b - ln c`，
且当且仅当 `ρ` 常数时取等号。

严格性风险点（需要在论文里写清）：
1. `E(φ)` 是“proxy”，不是模型任务指标本身。
2. `E(φ) ≥ 1/I(φ)` 是假设（可由 CRLB 解释），但不是自动由 Transformer 注意力推出的定理。

### 3.2 “残差 O(1/ln b)”的全域说法不成立（尤其 b=500K）

原因：
1. Ci 的小参数展开只在 `x≪1` 成立，`φ→0` 时 `b^{-φ}` 不小。
2. 对角 ridge 的 δ 近似在 φ-space 的宽度不是单纯 `O(1/ln b)`，会混入 L、Jacobian 等因素。

建议写法：
1. 只说“mid-band 捕获度高（R²_mid>0.99）”，把残差来源归因于 boundary + ridge 有限宽度 + 离散化。
2. 避免任何“pointwise residual 很小 / o(1)”的措辞。

当前修正：
已在 [CORE_THEORY.md](./CORE_THEORY.md) 删除 “O(1/ln b)” 的硬上界措辞，改成“有限宽度导致的 δ 近似误差”。

### 3.3 “β·min 是余弦积分渐近精确解，残差 O(b^{-γ})”过强

更安全的表述：
1. `β·min` 来自 bulk + off-diagonal 下的 Ci 小参数展开（典型 β≈1/2）。
2. 剩余项规模通常是 `O(1/ln b)` 叠加 boundary 项，而不是 `O(b^{-γ})` 这种指数级衰减。

当前修正：
已在 [CORE_THEORY.md](./CORE_THEORY.md) 将该句改写为“bulk 主导项 + 残差 O(1/ln b)+边界项”的区域性结论。

这条 scaling law 目前是强实验支持，但推导依赖对 `α(L,b)` 的校准假设。
