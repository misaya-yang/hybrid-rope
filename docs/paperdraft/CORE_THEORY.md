## 1. 推导链（6 步）

```
D(Δ) 距离先验
  → 相位碰撞核 K(φ₁,φ₂) = ∫D(Δ)cos(b^{-φ₁}Δ)cos(b^{-φ₂}Δ)dΔ
  → Broadband 投影: K ≈ αδ(φ₁-φ₂) + βmin(φ₁,φ₂)     [算子: αI + βA⁻¹]
  → 变分泛函: J[ρ] = (α/2)∫ρ² + (β/2)∫∫ρρmin - μ∫ρb^{-2φ}
  → Euler-Lagrange ODE: ρ'' - τ²ρ = γb^{-2φ},  τ=√(β/α)
  → 通解: ρ* = C₁cosh(τφ) + C₂sinh(τφ) + Pb^{-2φ}
  → CDF 反演: φ_k(τ) = 1 - (1/τ)arcsinh((1-u_k)sinhτ)
```

**唯一核心近似**：Broadband 投影（步骤 2→3）。残差 O(1/lnb)，R²>0.99。其余全是精确推导。

## 2. 三个定理

**Theorem 1**（ODE 精确解）: 通解由 hyperbolic tether（cosh/sinh）+ Fisher pulse（b^{-2φ}）两项竞争构成。τ 控制平衡。

**Theorem 2**（渐近退化）: τ→0 时 EVQ 光滑退化为 geometric RoPE。Geometric 是 EVQ 族的 τ=0 特例，不是竞争对手。

**Waterbed 不等式**: ∫ln E(φ)dφ ≥ lnb - lnc。Jensen 等号条件为 ρ≡1（geometric）。Geometric 最小化全局对数误差体积，但误差分布不均匀（E ∝ b^{2φ}，低频段指数爆炸）。EVQ 使误差趋于均匀但总体积增大。

---

## 3. Broadband 近似的算子理论地位

min(φ₁,φ₂) 是 A = -d²/dφ² 在混合边界条件下的 Green 函数。因此：

**K_approx = αI + βA⁻¹**（Identity + Resolvent）

这不是经验拟合，是精确核在二参数算子族上的 Hilbert-Schmidt 最优投影（Galerkin 投影）。

**为什么 R²>0.99 但全矩阵残差 35-49%**：R² 衡量中间频率区（渐近物理区），残差来自三个边界效应：(1) UV 边界 φ→0 的离散化误差；(2) IR 边界 φ→1 的波长超限；(3) δ 函数的物理脊宽 O(1/lnb)。

**Power-law 先验下更好**：D(Δ)∝1/Δ 时，βmin 不是经验拟合而是 Ci 函数的渐近精确解。结构残差降至 O(b^{-γ})，但脊宽误差仍是 O(1/lnb)。

## 4. Fisher → 注意力效用的理论桥梁

**Laplace 桥**：纯 RoPE 贡献的注意力 logit K(Δ) = ∫ρ(φ)cos(b^{-φ}Δ)dφ。对 Δ=0 做二阶展开：

K''(0) = -∫ρ(φ)b^{-2φ}dφ = -𝓗（即泛函中的 Fisher 项）

Softmax 的 Laplace 近似：A(Δ) ≈ exp(-𝓗Δ²/2τ)。**Fisher 项 = 局部注意力高斯分布的精度矩阵。**

**成立条件**：(1) 微扰极限 Δ→0；(2) 全局无假峰（无空间混叠）。

**失效区**：大 Δ 时高频 cos(ωΔ) 产生周期性假峰（spatial aliasing）。Fisher 只看局部曲率，看不到远处的混叠。

**修正方案（可选 Remark）**：用 Expected Contrastive Margin 替代 Fisher：
U(ω) = E_Δ[1-cos(ωΔ)] = 1-sinc(ωL)

- 低频：U ≈ ω² → 退化为 Fisher（向后兼容）
- 高频：U → 1（饱和，消除紫外灾难）

## 5. cosh 族为什么 1 参数打赢 32 参数

**n-width 论证**：泛函 J[ρ] 严格凸，有唯一全局解析解 ρ*。min 核的谱特征值 λ_k ~ O(k⁻²)（快速衰减），Kolmogorov n-width 衰减极快。

DAPE 的 32 自由度优化 = 用离散经验测度逼近连续最优解。1D 平滑分布的 CDF 反演离散化误差 W₂ ~ O(N⁻¹)，泛函二阶展开 → ΔJ ~ O(N⁻²)。

**N=32 时 O(N⁻²) ≈ 0.1%**。cosh 族已捕获 >99.9% 的泛函物理方差。DAPE 多出的 31 个参数只是在拟合离散化噪声。

**密度比下界**：ρ(0)/ρ(1) ≥ cosh(τ)。由 ODE 边界条件 + 极值原理严格证明。任何满足泛函约束的最优分配都必须具有至少这种量级的高频偏置。

## 6. Waterbed 的精细结构

**PPL 看不到 waterbed 的原因**：高频误差上升仍在 Softmax 的过参数化冗余区间内。PPL 是 token-level 全局平均，对频率轴的 aggregation 把 waterbed 效应掩盖了。

**下游任务看到 waterbed 的原因**：不同任务是频率轴上的带通滤波器。Retrieval = 低通（依赖远距离寻址，低频段误差被压低 → 提升）；Multi-hop = 中高通（依赖局部逻辑链精确性，高频容量被抽走 → 退化）。

**Waterbed 的量化**：偏离 geometric 的额外水床体积 = D_KL(Uniform || ρ)。加权 L² 误差 ≥ χ²(U||ρ)/c² ≥ D_KL/c²。

**跨学科同构**：
- Waterbed ≡ 控制论 Bode 灵敏度积分定理
- Geometric ≡ 信号处理 Constant-Q 小波变换（尺度不变）
- EVQ 偏移 ≡ 从 Constant-Q 向 Gabor 变换过渡 → Heisenberg 时频不确定性

## 7. τ* Scaling Law（核心理论贡献）

### 7.1 Conjecture: Inverse Square-Root Law

$$\tau^*(L_{train}) = \frac{d_{head}}{\sqrt{L_{train}}}$$

**实验验证**（3 个独立数据点）：

| L_train | 预测 τ* = 64/√L | 实测 τ* | 误差 |
|---------|-----------------|--------|------|
| 128 | 5.66 | >5.0 | ✓ |
| 1024 | 2.0 | ≈2.0 | 0% |
| 2048 | 1.414 | ≈1.5 | ~6% |

**待验证预测**：L=4096 → τ*=1.0（Phase 8 数据可验证）。

**常数 C = d_head = 64 的含义**：频率空间维度（head_dim/2=32 个通道，但 d_head=64 是嵌入维度）。可通过换 head_dim=128 的实验验证 C 是否跟着变。

### 7.2 变分推导

总逼近误差分解为两个对立惩罚：

E(τ) = β*/τ + (α₀*·L)·τ

- β*/τ：局部截断误差（高频不够 → 近距离分辨力差），与 L 无关
- α₀*·L·τ：长程混叠误差（高频过多 → 远距离产生假峰），与 L 线性增长

最小化：τ* = √(β*/(α₀*·L)) ∝ 1/√L

**物理意义**：L 增大时，防混叠的低频需求线性激增，频率分配质心以 1/√L 速度向低频退移。

### 7.3 模型大小无关性（几何解耦）

Transformer 训练 = 双层优化：
- 内层（Q/K 权重）：在给定频率基底 V_τ 上做最优正交投影（软分配）
- 外层（τ）：选择最优的频率子空间几何（硬分配）

由正交投影定理，外层的几何截断误差 = ||(I - P_{V_τ})·Target(L)||²，只依赖目标分布拓扑（L）、空间维度（d/2=32）和基底分布（τ）。模型参数量 N 被完全边缘化。

**条件**：模型不处于极度欠拟合（50M 足以做 32 维线性组合）。

### 7.4 信息论解释（贝叶斯相变）

- 短 L（128）：低频通道梯度 SNR ≈ 0，Q/K 学不到长程 pattern → 需要强先验（大 τ）阻止低频过拟合噪声
- 长 L（2048）：长距离观测对 O(L²) 爆炸 → 数据证据充足，Q/K 自己能学好 → 先验退火（小 τ），频率分配交还给梯度下降

**τ* scaling law 描述的是从 "硬先验主导" 到 "数据驱动主导" 的贝叶斯相变临界方程。**

### 7.5 ⚠️ 谨慎声明

- 仅 3 个数据点拟合，自由度接近 0。需要 L=256/512/4096 的验证点
- C=d_head 可能是巧合。需要不同 head_dim 实验确认
- 推导中 "混叠误差 ∝ L·τ" 的线性假设需要更严格的证明
- 论文中建议写为 Conjecture + 实验支持，不要写成 Theorem
